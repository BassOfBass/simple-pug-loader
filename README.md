## Install

[npm][]:

```sh
npm install better-pug-loader
```

[yarn][]:

```sh
yarn add better-pug-loader
```


## Usage

```js
const BetterPugLoader = require('better-pug-loader');

const betterPugLoader = new BetterPugLoader();

console.log(betterPugLoader.renderName());
// script
```
